# Antvengers – Control de Gastos

> Aplicación web para la gestión y control de gastos personales, desarrollada con React.

---

## Propósito del proyecto

**Antvengers** es una aplicación web orientada al control y gestión de gastos personales, desarrollada como parte de un ejercicio académico. Su objetivo es permitir a los usuarios:

- Registrar ingresos y gastos
- Visualizar su información financiera
- Organizar sus consumos por categorías
- Mejorar sus hábitos financieros

El proyecto también busca aplicar buenas prácticas de desarrollo frontend usando React, incluyendo componentización, manejo de estado, consumo de APIs y almacenamiento local.

---

## Tecnologías utilizadas

| Tecnología | Descripción |
|---|---|
| React | Librería principal para la construcción de la interfaz |
| Vite | Herramienta de desarrollo rápida para proyectos React |
| React Router DOM | Manejo de navegación entre páginas (SPA) |
| SweetAlert2 | Alertas modernas para interacción con el usuario |
| Tailwind CSS | Estilos rápidos y responsivos |
| JavaScript (ES6+) | Lógica de la aplicación |
| LocalStorage | Persistencia de datos en el navegador |
| Fetch API | Consumo de servicios backend |

## Instalación y ejecución

Sigue estos pasos para ejecutar el proyecto en tu entorno local:

### 1. Clonar el repositorio

```bash
git clone <https://github.com/MarlonMaca2022/AntvengersFront>
cd antvengers-react
```

### 2. Instalar dependencias

```bash
npm install
```

### 3. Ejecutar el proyecto

```bash
npm run dev
```

### 4. Abrir en el navegador

```
http://localhost:5173
```

## Funcionalidades principales

- **Autenticación** – Registro e inicio de sesión de usuarios
- **Dashboard** – Visualización de gastos e ingresos
- **Registro de gastos** – Agregar nuevos movimientos financieros
- **Configuración de usuario** – Gestión del perfil
- **Consumo de API** – Integración con servicios backend
- **Persistencia de sesión** – Datos almacenados con localStorage
- **Alertas interactivas** – Notificaciones con SweetAlert2
